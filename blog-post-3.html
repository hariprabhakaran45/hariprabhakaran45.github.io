Virtual Try-On for E-commerce

Published: April 2025

My recent project combines Node.js backend with VITON‑HD to create a seamless virtual try-on experience for fashion e-commerce. Users can upload their photo and overlay clothing items in real-time.

The core stack includes:

  • Node.js & Express: Handles file uploads, user sessions, and API integration.
  • VITON‑HD: A deep-learning model enabling realistic try-on overlays.
  • Frontend: Simple interface for uploading images and viewing results. Styled with Bootstrap/CSS.

To run it locally: 

npm clone https://github.com/hariprabhakaran45/E-Commerce-With-virtual-Trial-Experience.git
cd E-Commerce-With-virtual-Trial-Experience
npm install
node server.js

Once running, navigate to http://localhost:3000/try-on, upload your photo, and see the virtual try-on magic in action!

View the full source code and README here: GitHub Repository

In future versions, I plan to integrate user authentication, cloud deployment (e.g. AWS/Heroku), and save try-on outputs so users can download/share them.
